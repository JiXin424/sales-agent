# 如何找到目标客户/工会主席

> **核心目标**：联系上企业工会主席（或关键决策人），建立有效沟通。
> **常见挑战**：不知道工会主席姓名、找不到直接联系电话、企业前台拒绝转接。

## 获取工会主席联系方式的实战方法

### 方法一：通过官网与新闻锁定主席姓名
销售人员可以首先访问目标企业的官方网站，在“企业文化”“党群建设”“工会活动”等栏目中搜索工会相关信息。许多企业在报道工会活动时会直接出现工会主席姓名。你也可以通过百度等搜索引擎，使用“企业名称+工会主席”组合词查找公开新闻。拿到姓名后，致电企业总机，直接说出“请转工会张主席”，前台转接的成功率将显著提升。

> **用户可能搜**：怎么查公司工会主席名字？ / 网上如何找工会主席？ / 怎么让前台转接工会？

### 方法二：拨打财务等部门电话迂回获取工会联系方式
如果总机拒绝转接，销售人员可以尝试拨打目标企业的财务部、行政部或人力资源部电话（通常比总机好打）。接通后，礼貌说明需要对接工会事宜，请对方告知工会直拨电话或帮忙转接。财务等部门人员对非业务电话戒备较低，往往愿意提供内线。

> **用户可能搜**：总机不给转工会怎么办？ / 怎么绕过前台找工会？ / 打哪个部门电话能要到工会电话？

### 方法三：全渠道搜索工会相关关键词
确定目标企业后，销售人员应多渠道挖掘联系人和联系方式。利用小红书、抖音、企业官网、微信公众号等平台，搜索“工会”“党群”“党办”“工会主席”“某某主席”“某某领导”等关键词。同时可以关注阶段性党政热门词汇，如“十五五规划”“职工之家”“劳模评选”等，这些内容往往由工会主导发布，文末可能留有联系人信息。

> **用户可能搜**：小红书上怎么找工会主席？ / 抖音能搜到工会联系人吗？ / 公众号找工会主席 / 十五五规划工会联系人

### 方法四：从媒体文章的记录人或编辑找到对接人
在企业相关媒体发布的文章（如企业内刊、行业媒体报道）底部，通常会有“记录人”“编辑”“通讯员”等署名。这些角色往往是企宣或工会干事，是极佳的对接入口。销售人员可以通过公司总机找到这些人，再请其引荐工会主席。

> **用户可能搜**：文章底部的通讯员是谁？ / 怎么通过新闻找到企业联系人？

### 方法五：利用官方招投标信息挖掘客户
定期浏览政府采购网、企业官网“招采平台”等公开渠道发布的员工福利、慰问品、电影票等采购公告。从招标信息中可以明确知道哪些企业有需求，并能从公告或往年中标公告中获取联系人及联系电话。优点是信息易得、需求明确；缺点是竞争对手也容易看到，导致竞争激烈。

> **用户可能搜**：哪里看工会的招投标信息？ / 怎么从招标信息里找客户电话？ / 招投标找客户好用吗？

### 方法六：从竞争对手的招标信息中“截胡”
除了直接找客户，还可以关注主要竞品（同业福利平台）的公开中标信息。中标公告中一般会公示采购方名称，这些正是已经有福利采购预算的企业。你可以将其列为高意向客户进行跟进，研究其痛点，提供更优方案。

> **用户可能搜**：怎么从竞争对手手里抢客户？ / 竞品中标信息在哪里看？

### 方法七：借上下游子公司或兄弟单位名义突破
如果直接联系目标企业困难，可以向其上下游子公司或兄弟单位下手。以“集团供应商”或“有合作资料需要转交工会”等理由联系子公司，对方往往戒备更低。通话时可自然提及公司福利产品的优势，先让对方对福利话题产生兴趣，再请求转介绍或转接。

> **用户可能搜**：怎么通过子公司找总部工会？ / 以供应商名义找工会管用吗？

### 方法八：盲拨行政办公室，用福利话题吸引转接
在没有准确姓名和电话的情况下，可以直接拨打行政办公室或其他综合部门。接通后，以真实身份坦诚说明：“我是做员工福利服务的，有些新品可能对你们单位的福利采购有帮助，想跟工会负责人沟通一下。” 因为福利话题与接听者个人利益相关（大家都会关心节日发什么），对方更容易产生兴趣并帮忙转接。

> **用户可能搜**：盲打说什么能让对方转接？ / 怎么用福利话题吸引前台？ / 行政办公室电话怎么打？

### 方法九：假借上级主席推荐接触下属
如果接触到的是工会主席的下属（如工会干事），而非主席本人，销售人员可以说：“是上级工会（如区总工会/市总工会）的王主任推荐我来联系贵单位工会的，想做个服务对接。” 这种“上级背书”往往能让下属打消疑虑，推动后续沟通。

> **用户可能搜**：找到工会干事怎么让他引荐主席？ / 借上级工会的名义怎么说？

### 方法十：使用“钉钉企典”精准查询企业电话
钉钉企典是钉钉内嵌的企业信息查询工具，数据来自于工商公开信息和企业自行维护的资料，精准度较高。销售人员可以在钉钉中直接搜索目标企业名称，查看公开的联系方式甚至组织架构，有时能直接找到工会办或综合办的联系电话。

> **用户可能搜**：钉钉企典是什么？ / 怎么用钉钉查企业电话？ / 钉钉找客户准不准？

### 方法十一：善意谎言策略，以总工会名义联系
在某些难以突破的场景下，可以尝试以“省/市总工会工作人员”的身份致电企业，就某项工会工作（如职工福利调研、政策传达等）进行沟通。这种方式利用了总工会的权威性，能够快速越过前台。但需注意：应准备充分的业务说辞，避免轻易拆穿，同时确保后续转入正常商务沟通。

## 销售心态心法：成为被需要的人

### 重构认知：你不是推销员，而是价值提供者
在拿起电话之前，销售人员必须完成内心角色的切换。你并不是在“找客户”“销售产品”，而是在“成为被他人需要的人”。你正在帮助工会主席解决员工福利品采购的合规性、丰富性和便捷性难题，是在为他们创造实实在在的价值。这一认知的转变，能有效消解你内心的抗拒和恐惧，让你在通话中更加自然和自信。

> **用户可能搜**：销售心态怎么调整？ / 克服推销恐惧 / 工会销售心理建设 / 不想推销怎么办？

### 接受过程：销售是概率游戏，但你可以提高胜率
必须清醒地认识到，销售本身就是一场“概率游戏”，但你有机会通过专业动作提高成功概率。养成记录的习惯：把每一次被拒绝的原因记下来，复盘话术、优化渠道和切入时机。同时，不要只盯着成交这一个目标，要庆祝每一个微小的胜利——例如“今天成功加了2个微信”，在过程中持续给自己正向反馈。

> **用户可能搜**：被拒绝多次怎么坚持？ / 销售怎么有好的心态？ / 如何提高销售成功率？

### 能量管理：保持高频行动，避免情绪内耗
销售工作拼的不是一时的激情，而是持续的能量。坚持高频次的行动，但决不让负面情绪长期占据内心。日常养成三个习惯：第一，与团队成员共享成功案例，互相激励，为彼此注入能量；第二，遇到挫折时，找一个安全的发泄出口，比如与同事或好朋友吐槽释放；第三，规律作息与锻炼，保证身体这一能量基础。

> **用户可能搜**：销售没动力怎么办？ / 如何管理情绪？ / 天天打电话很丧怎么调整？

### 长期主义：每一步都算数，信任需要时间沉淀
个人业绩和客户信任的建立，都不是一蹴而就的。即使本次电话没有成交，只要你持续提供行业观点、福利选品趋势等有价值的信息，就能在客户心中留下印记，为下一次成交积累势能。要知道，工会主席的圈子很小，口碑传播可能比广告更快。量变终将引起质变，当你坚持不懈到一定程度，结果一定不会差。

> **用户可能搜**：长期没业绩还要坚持吗？ / 工会圈子如何做口碑？ / 销售怎么建立信任？

### 坚持、自信和决心：用真诚和专业抓住每一个时机点
前期的所有苦功夫都不会白费。央国企工会的福利供应商更换周期通常在2-3年左右；在周期内持续跟进、渗透，等到更换供应商的关键节点到来时，你前期积累的关系和信任会瞬间爆发，成交就是水到渠成。有时候客户暂时没有采购需求，但未来一定有；有时候你的产品和竞品看起来差不多，但你的真诚、多次拜访展现的责任心，会成为打动工会主席的决定性因素。坚信，只要坚持并有决心，就一定会有结果。

> **用户可能搜**：央国企工会多久换一次供应商？ / 客户一直说没需求怎么办？ / 产品没优势怎么赢？ / 靠真诚能赢单吗？

## 常见问题（FAQ）

### 关于找到联系人
**Q1：怎么找到工会主席的联系电话？**
A：可通过官网新闻、招投标公告、企业微信公众号等渠道先获取主席姓名；再致电总机或使用钉钉企典找到企业公开电话，直接要求转接。详细步骤参见上文“方法一”至“方法六”。

**Q2：不知道主席名字，盲打时第一句话怎么说？**
A：可以拨打行政办公室等部门，开门见山说明自己是做员工福利服务的，有新福利方案希望能与工会负责人沟通。强调与接听者个人利益相关的福利点，更容易引发兴趣并获得转接。

### 关于突破电话障碍
**Q3：前台死都不转接，还有其他办法吗？**
A：可转打财务部、人力资源部等非一线部门的电话，这些部门的拒接率往往更低。或者从上下游子公司、兄弟单位以“送资料”“供应商对接”等名义找内部员工帮忙转接。

**Q4：以总工会名义打电话管用吗？怎么说？**
A：这是一种有效的突破手段，利用了上级工会的权威性。通话时可以就某项工会工作（如职工福利调研）进行了解，迅速建立通话合法性。但需提前准备好说辞，并注意后续自然过渡到正常商务沟通。

### 关于工具与渠道
**Q5：钉钉企典怎么用？查出来的电话准吗？**
A：在手机钉钉内搜索“钉钉企典”应用或直接搜索目标企业名称，即可看到该企业基于工商和自维护的公开信息，包括联系电话。精准度较高，尤其适合获取国企、大型企业的对外公开号码。

### 关于销售心态与坚持
**Q6：打了一天电话都没进展，心态崩了怎么办？**
A：首先要接受销售是“概率游戏”，没有谁可以每天都成交。记录被拒原因并优化话术，同时转向庆祝微小成功（比如今天比昨天多说了两分钟）。感到沮丧时可以和同事吐槽排解，然后继续行动。

**Q7：跟进很久的客户一直说没需求，还要坚持吗？**
A：央国企工会的福利供应商更换周期多为2-3年，现在没需求不代表未来没有。你的任务是保持每月1-2次有价值的内容互动（如新的福利政策、选品案例），持续刷“存在感”，当客户有需求时第一个想到你。

**Q8：产品功能跟竞争对手差不多，怎么打动工会主席？**
A：产品和价格趋同时，差异化就在服务和你本人。通过真诚的态度、多次拜访展现的责任心，以及持续的专业跟进，让主席认为“交给你最放心”。工会主席圈子小，口碑远比功能更重要。
