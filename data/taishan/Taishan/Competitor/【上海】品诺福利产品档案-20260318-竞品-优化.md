# 一、 品福通（综合福利卡）

品福通是品诺福利推出的通用型综合福利卡，它将蛋糕、电影、商城商品、在线点餐等多种消费场景集成在一张卡上。客户购买后发放给员工，员工可自主选择喜欢的福利内容进行兑换。

## 1.1 品福通各子产品详解

为了满足不同客户的预算和需求，品福通细分出多个产品型号（简称“子产品”），它们在包含的消费资源和销售政策上各有侧重。以下是各主要子产品的自包含描述：

### 品福通-A
- **核心资源：** 蛋糕 + 电影 + 商城全品类（具体包含：商品、图书、演出、旅游、健康等）+ 在线点餐（肯德基、麦当劳、奈雪的茶）。
- **交付平台：** 品诺商城 或 品倍福。
- **平台资源差异：** 当通过**品倍福**平台交付时，在品诺商城资源的基础上，额外增加对汉堡王、霸王茶姬、哈根达斯、瑞幸咖啡、库迪咖啡等5项餐饮资源的支持。
- **本品类特殊子产品：**
  - **品福通-A (a)：** 电影票价与猫眼、淘票票等票务平台的非活动页面常规售价基本持平。
  - **品福通-A 减：** 这是一种商城资源可定制的版本。您可以按需仅开通商城中的部分商品类别。系统支持按【全量商品】、【自营商品】或【优势商品】三种类型进行勾选，但无论选用哪种，最少都需要勾选6个商品分类。

### 品福通-B
- **核心资源**：蛋糕 + 电影 + **商超券** + 在线点餐。**（特别注意：此版本不包含商城商品、图书、演出、旅游、健康等资源）**。
- **交付平台**：品诺商城 或 品倍福。本次新增的8项餐饮资源在品倍福平台均可使用。
- **本品类特殊子产品：**
  - **品福通-B (a)：** 该版本仅支持蛋糕、电影、商超券三项核心资源的兑换。

### 品诺优选
- **核心资源**：专注于**商城**内资源，具体包含商品、图书、演出、旅游、健康等。
- **注意：** 本品诺优选**不包含**蛋糕、电影和在线点餐资源。
- **交付平台：** 品诺商城 或 品倍福。

### 品福通-Plus
- **核心资源**：品福通-A的全部资源 + 额外的**商超券**。可以理解为“品福通-A”的加强版。
- **交付平台：** 品诺商城 或 品倍福。
- **平台资源差异：** 与品福通-A逻辑一致，仅在**品倍福**平台交付时，才包含新增的汉堡王、霸王茶姬等5项餐饮资源。
- **合作门槛：** 北京、上海、广州、深圳四个一线城市的客户，单笔开卡订单需达到**10万元**；其他城市的客户，单笔开卡订单需达到**5万元**。

### 优选 SE
- **核心资源**：仅开放**商城商品**版块。这是品诺优选的简化版。
- **交付平台：** 品诺商城 或 品倍福。

## 1.2 品福通销售提成政策与计算案例

品诺销售的提成收入与最终售价（通过“配赠比例”调节）直接相关。**配赠比例**指为了促进成交，您在客户支付的金额之外，额外赠送的点数比例。赠送的比例越高，您的提成比例相应会降低。

**提成计算示例：**
我们以售卖100元的“品诺优选”产品为例。如果成交时，您承诺了1%的配赠比例，那么最终充值到客户卡里的实际点数是101点。此时，您的这笔订单提成金额为：100元 * 6.0% = **6元**。（注：此提成计算为税前，下同）。

关于各子产品在不同配赠比例下的具体提成比例，以及**预付**和**实结**两种结算方式下的提点差异，请参考公司内部发布的《品福通销售政策明细表》。

## 1.3 品福通重要注意事项
- **卡券有效期**：所有品福通系列卡券的默认有效期为**2年**。
- **溢价销售**：品福通、品诺优选系列产品，**不允许**以高于刊例价的价格进行销售。
- **卡券定制**：公司提供的标准交付物是统一卡面和券样。如果客户有定制卡券的需求，请务必查阅并遵循公司发布的《定制卡券套及印刷品收费标准》进行操作。

# 三、 电影系列产品

品诺电影产品线是客户解决“电影预算”的专项方案，主要分为无需选片的“点卡”和固定张数的“次券”两大类。

## 3.1 电影点卡：灵活的电影钱包

### 3.1.1 电影点卡子产品详解
电影点卡就像一个专门用于观影及相关消费的数字钱包，不同型号在包含的资源上有所不同：
- **品诺电影-A (a) 系列**：电影票价与猫眼、淘票票等票务平台的非活动页面常规售价基本持平。
- **品诺电影-A 减版本**：这是一个可以对商城资源进行定制的版本。其商品版块开通规则与“品福通-A减”一致，即至少需要勾选6个分类。
- **品诺电影-B 与 B+版本**：这些版本不含商城商品、图书等。其中，**品诺电影-B+**额外包含了“影音会员”等虚拟商品。同样，只有在**品倍福**平台开通时，才支持拓展的在线点餐资源。

### 3.1.2 点卡销售提成政策与计算案例

**提成计算示例：**
以销售一款100元的“品诺电影-A”点卡为例。若您以1%的配赠比例成交，客户获得101点，您的提成金额为：100元 * 6.4% = **6.4元**。
**特别注意**：此通用政策不适用于“品诺电影-B”的特殊版本（即配赠比高达30%、35%、40%的版本），它们有独立的结算规则。

各子产品在不同配赠下的具体提成点数，请查阅内部《电影点卡销售政策明细表》。

### 3.1.3 电影点卡注意事项
- **溢价销售**：电影点卡产品，**不允许**溢价销售。
- **卡券有效期**：默认有效期为**1年**。
- **卡券定制**：常规提供标准卡券。若有定制需求，需遵照《定制卡券套及印刷品收费标准》执行。

## 3.2 电影次券：一券一影的兑换券

### 3.2.1 次券产品详解与使用规则
电影次券是一种按“张”兑换的服务，通常1张次券可兑换1张电影票，或用于抵扣等额的商品/图书。按使用范围分为：
- **本地通**：仅限于某个省/市使用，如“北京本地通”仅限北京市，“浙江本地通”仅限浙江省内。
- **全国通**：支持全国范围使用。其销售提点是在同档位“本地通”政策的基础上打80折。

**核心使用规则（务必向客户说明清楚）：**
1.  **兑换范围**：除【电影-全国通-65】档位可兑换电影、商品、图书、蛋糕、演出、游乐、健康外，其他普通档位仅可兑换**电影、商品、图书**。每张券仅可使用1次。
2.  **补差价与合并支付**：兑换电影票时，如果影票价格高于券面值，**仅支持单张支付并补差价**。若用2张券合并支付，则不支持补差价，可能导致券内余额浪费。
3.  **特殊提点规则**：如果客户定制的次券被限定为“仅限电影”或“电影+影音”，那么您的提成点数为本地通提点的**50%**。如果同时满足了“全国通”和“仅限电影”两种限制，则在“全国通”提点基础上再打五折。若折算后的提点低于0.7%，则按0.7%保底计算。

### 3.2.2 次券销售提成政策与溢价说明
**溢价销售：** 电影次券是**允许溢价销售**的。但要注意，售价不能超过该档位的上限。溢价部分的金额，其85%将作为额外的提成给您。

**溢价提成计算示例：**
假设您成功将一个80档电影次券以80元的价格售出，提成计算拆分如下：
- **标准提成**：刊例价58元 * 7%的标准提成 = 4.06元
- **溢价提成**：(实际售价80元 - 刊例价58元) * 85% = 18.7元
- **您的总提成**：4.06 + 18.7 = **22.76元**

各档位在不同售价下的具体提成点数，请查阅内部《电影次券销售政策明细表》。

### 3.2.3 电影次券注意事项
- **卡券有效期**：默认为**1年**。
- **卡券定制**：如有定制需求，遵照《定制卡券套及印刷品收费标准》执行。

## 3.3 品享电影：独立的轻量级平台
品享电影是一个独立的交付平台，产品形态为次券，通过“品享电影”服务号交付给客户员工。

### 3.3.1 品享电影子产品说明
- **品享电影（影音）**：可兑换**电影票和影音会员**（如爱奇艺、QQ音乐等）。兑换规则与点卡/次券类似，支持补差价和合并支付。
- **品享电影（仅电影）**：只能兑换电影票，分为“全国通”和“本地通”系列。仅限在“品享电影”独立平台使用。

### 3.3.2 品享电影提成与溢价政策
1.  **提点规则**：全国通的提点是本地通的80%。影音版和仅电影版的提成政策一致。若折算后提点低于0.7%，则按0.7%计算。
2.  **溢价销售**：品享电影次券**支持溢价销售**，但售价不得高于相应档位的上限。

品享电影各档位的具体刊例价、底价和提成比例，请查阅内部《品享电影业务提成政策明细表》。

# 四、 品倍福：企业员工福利管理系统

# 六、 年节关爱：礼包与定制化服务

年节关爱产品线是品诺服务于春节、端午、中秋等传统节日员工慰问的核心业务，核心交付物是“礼包”或“单品”。

## 6.1 关键概念解析：礼包、单品与交付
为避免混淆，对几个关键术语解释如下：
- **年节礼包**：将多个商品组合成一个固定档位的套餐，性价比高。员工兑换时一次用完一张券。礼包编号后缀为“R”的是常规款，后缀为“Y”的是引流款。
- **年节单品**：客户员工获得一张储值卡/券，可以在“单品专区”里像逛超市一样，自由选择商品，卡内余额可分多次消费，直至用完。
- **常规与定制**：
  - **标准礼包**：公司统一定价、统一内容的现成礼包。
  - **常规定制**：在现有标准礼包和商品库内，为客户进行“加减法”式的搭配和挑选。
  - **非常规定制**：需要从现有商品库**之外**进行采购并组包。其成本较高，且数量不能超过整体礼包数的20%。
- **交付方式**：主要有两种。
  - **线上专区**：客户员工通过H5页面或品倍福系统，在线选择并下单，由物流快递到家。
  - **线下配送**：将所有礼包/商品通过物流专车集中运送至客户指定的一个或多个地点，或根据地址分别快递给每位员工。

## 6.2 年节产品销售政策精要
- **提成与折扣挂钩**：年节礼包的提成比例与您给客户的“销售折扣”直接相关。销售折扣越低，您的提成点也相应越低。线上和线下有各自的提成政策表，请查阅内部文件。
- **溢价销售**：标准礼包和定制礼包都**允许溢价销售**。标准礼包溢价上限是刊例价的25%，定制礼包是20%。溢价部分的85%将计入您的个人提成。
- **定制门槛**：不同复杂度的定制需求，有明确的订单金额门槛：
  - **简约摘选定制（线上/线下）**：订单金额需≥2万元。
  - **库内重组定制（线上/线下）**：订单金额需≥5万元。
  - **库外重组定制（线上/线下）**：订单金额需≥30万元。

## 6.3 年节关爱重要注意事项
- **定价依据**：在为常规定制和非常规定制礼包选品时，**不可以**参考“品优选”前端售价作为礼包档位的刊例价。
- **定点配送政策**：线下配送需提前申请，周期约5-7个工作日。严禁销售私自将配送订单转为线上卡券发给客户代下单，一经发现将面临取消提成和高额罚款。
- **过期处理**：年节礼包一旦过期，**只能换卡，不能延期**。换卡可转为当期在售的同档位节日礼包，有效期默认为3个月。

请您务必理解并遵守以下贯穿所有业务的通用规则，这是您日常工作的底线和依据。

## 11.1 提成核算基础：标准个人提成
- **业绩门槛**：提成核算并非一开单就按高标准执行。只有**同一客户、同一产品、同一预算、同一结算方式**下，一个自然年内的系统订单，且回款金额累计达到**3万元**后，才能按“标准个人提成比例”核算。在达到3万元业绩之前，您的所有提成都暂按标准提成的**80%**计算。

## 11.2 实结业务与问题订单
- **实结业务**：指按客户实际消费金额结算并回款的业务。除品享福、品倍福餐补等有明确实结政策的产品外，其他产品如果要做实结，必须经集团总裁特批，且批下来后的提成比例统一为极低的0.7%。
- **问题订单**：这是绝对的红线。凡是发现订单或客户信息弄虚作假，或者因个人操作给公司造成损失的，不仅单笔提成全扣，还要赔偿公司损失，相关管理层也会被追责。

## 11.3 激励政策：2025年大额订单激励
为了鼓励大家冲击大单，公司对2025年4月21日之后新签约的客户推出激励政策：
- **激励标准**（单个客户连续三个月累计开卡并回款的业绩）：
  - **北上广深**：累计≥**80万**，一次性奖励**5000元**。
  - **成都、武汉、杭州等连续三个月**：累计≥**60万**，一次性奖励**3000元**。
  - **重庆、郑州、昆明等**：累计≥**40万**，一次性奖励**2000元**。
- **注意**：同一客户只奖励一次。此政策的试行期已延长至2026年3月31日。部分特定产品（如餐补系列、渠道订单等）不参与此激励。

A：品福通是综合福利卡，根据型号不同，可以兑换蛋糕、电影、商城商品（如米面粮油、小家电）、图书、演出票等。品福通-A和Plus版资源最全，品福通-B版不含商城商品，但有商超券。

**Q2：客户想把电影次券卖贵一点，我该怎么算我的提成？（用户也常问：电影次券可以溢价吗）**
A：可以溢价销售，但售价不能超过该次券的档位上限。您的提成由两部分组成：标准提成（刊例价*提点）+ 溢价提成（超出刊例价的部分*85%）。具体档位上限和刊例价请查阅产品政策明细表。

**Q3：我接了一个客户的饭补需求，应该给他推品倍福还是品享福？**
A：两者都专注餐补，但定位不同。品享福是专注餐饮的轻量级平台，合作门槛相对较低（一线城市年单≥80万）。品倍福-餐补是综合性更强的系统，但门槛极高（一线城市年单≥150万）。您可以根据客户规模和预算推荐。

### 关于销售政策与提成

**Q4：什么是配赠比例？我答应给客户多配赠，对我的提成有什么影响？（用户也常搜：配赠怎么影响提成）**
A：配赠是您给客户额外赠送的福利金额。例如配赠1%，客户付100元，能得到101点。但赠送的部分会冲减您的提成，所以**配赠比例越高，您的提成比例就越低**。开单时需权衡利弊。

**Q5：公司那个大额订单奖励5000元，有什么要求？**
A：这是针对2025年4月21日后签约的新客户激励。要求您在北上广深单个新客户的**连续三个自然月**开卡并回款累计达到**80万元**（其他城市门槛递减），可一次性获得5000元奖励。同一个客户只奖励一次。

**Q6：提成核算的“3万元门槛”是什么意思？**
A：指您和同一个客户，就同一产品、同一预算、同一结算方式下，一个自然年内的回款总额累计达到3万元后，后续该统计口径下的订单才能按“标准个人提成比例”核算。在此之前，您的提成暂按标准的80%计算。

### 关于系统与流程

**Q7：客户想做年节礼包，但想把里面的东西换几种，要走什么流程？（用户也常搜：年节礼包怎么定制）**
A：这属于“重组定制”需求。如果用的是公司商品库里的货，叫“库内重组”，需要订单金额达到**5万元**以上。如果是库外采购，属于“库外重组”，门槛是30万元。您需要通过OA发起礼包定制申请。

**Q8：赠送客户的VIP抵扣券，在OA里怎么操作？忘了在充值流程里加怎么办？**
A：赠送抵扣券**必须**和主卡在**同一个OA开卡/充值流程**中提交。如果忘了，未开通的可以重提，一旦流程已开通，则**无法补开**。请务必在提交流程前仔细核对。

**Q9：客户的蛋糕卡快过期了，能延期吗？**
A：蛋糕卡、电影卡、品福通卡过期后，您可以协助客户提交“过期延期”申请。常规最多延期**6个月**，超出6个月的特殊情况需要总经理审批。注意，年节礼包过期后不能延期，只能“换卡”。
